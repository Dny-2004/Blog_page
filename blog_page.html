Database Programming (DBP)

What is DBP?

Database Programming focuses on the creation, management, and interaction with databases using SQL and programming languages. It helps handle large volumes of data effectively.

Syntax Example

CREATE TABLE Students (
  ID INT PRIMARY KEY,
  Name VARCHAR(50),
  Marks INT
);

Uses

Used in banking, e-commerce, education, and any application requiring data storage and retrieval.

Example Query

SELECT Name, Marks 
FROM Students 
WHERE Marks > 75;

Java Programming

What is Java?

Java is a high-level, object-oriented language widely used for building applications from Android apps to enterprise systems.

Syntax

class Hello {
  public static void main(String[] args){
    System.out.println("Hello, Java!");
  }
}

Use

Used in web apps, Android development, and backend systems like Spring Boot. Known for reliability and platform independence.

Example Program

class Student {
  String name = "Dnynesh";
  void display(){
    System.out.println("Name: " + name);
  }
  public static void main(String[] args){
    Student s = new Student();
    s.display();
  }
}
